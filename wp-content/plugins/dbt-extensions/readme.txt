== Description ==

Allows you to install and manage custom and add-on fieldtypes and form processors for DB-Toolkit.

== Installation ==

1. Upload the plugin folder 'dbt-extensions' to your `/wp-content/plugins/` folder
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Select DBT Extentions from the menu group
